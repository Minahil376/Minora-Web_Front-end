# Minora Web – Fashion E-Commerce Frontend

Minora Web is a multi-page fashion e-commerce frontend built using HTML, CSS, and JavaScript.  
It demonstrates real frontend structure, product flow, and cart interface without frameworks.

## Features
- Home, Shop, Product, Blog, About, Contact, Cart pages  
- Product details with gallery, size, quantity, and add-to-cart UI  
- Cart with remove option, subtotal, total, and checkout section  
- Responsive design with reusable navigation and sections  

## Tech Stack
HTML5 • CSS3 • Vanilla JavaScript  

## Run Project
Open **index.html** in your browser  
(or use Live Server in VS Code)
